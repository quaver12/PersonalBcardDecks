h:Verbs!
--------
q:to earn
a:vierdienen
 - 
q:to rent
a:mieten
 - 
q:to stand
a:stehen
 - Used for objects that 'stand' on a table - e.g. Die Flasche steht auf dem Tisch. Not to be cofused with stellen
q:to lie (as in lie down not tell a lie)
a:liegen
- also used for objects that 'lie' on something - e.g. Das Buch legt auf dem Tisch. Not to be confused with legen - put
q:to put (something that stands)
a:stellen
 - not to be confused with stehen
q:to put (something that is lying down)
a:legen
 - not to be confused with liegen
q:to hang
a:hangen
 - Das Bild hangt am der Wand.

 - seperable! (washen ab)

h:Translate the following english phrases to german:
----------------------------------------------------
q:for example
a:zum Beispiel
 -
q:Los Gehts!
a:Let's go!
 - 
q:Drop it!
a:Lass das
 - Let go / drop it / stop doing that / oh come off it / oh please drop it. Could be used to command a dog to drop, could be used to tell someone to stop doing something.
