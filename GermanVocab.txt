h:Translate the following english phrases to german:
----------------------------------------------------
q:for example
a:zum Beispiel
 -
q:Los Gehts!
a:Let's go!
 - 
q:Drop it!
a:Lass das
 - Let go / drop it / stop doing that / oh come off it / oh please drop it. Could be used to command a dog to drop, could be used to tell someone to stop doing something.
